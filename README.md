# toy_car_auto_drive_by_yolo
Using yolo 8  for toy car auto drive  
使用yolo實作玩具車自動行駛  
這是個容易上手的版本  
使用直排輪的迷你三角錐當作引導物件
玩具車只有四個動作,往前,往左前,往右,停止.
使用樹莓3B當作影像來源
PC由藍芽操作玩具車


可分成4個單元  
1.圖片取得 get picture  
2.物件偵測 detect object  
3.決策 decision  
4.執行  execution
